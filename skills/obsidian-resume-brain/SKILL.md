# obsidian-resume-brain

A structured note system for resume source-of-truth data in Obsidian.

## What it does
- Provides consistent templates for profile, experience, projects, and job pipeline tracking.
- Encourages quantified, evidence-based bullet writing.
- Makes downstream resume tailoring easier and auditable.

## File layout
- `resume-brain/profile.md`
- `resume-brain/experience/_template.md`
- `resume-brain/projects/_template.md`
- `job-pipeline/_job-template.md`
