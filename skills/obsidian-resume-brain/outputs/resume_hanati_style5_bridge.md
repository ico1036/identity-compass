## 박사 연구: 물리학에서 퀀트로의 방법론적 전이

### LHC 양성자-양성자 충돌 데이터의 CNN 기반 신호/배경 분류

CERN LHC 실험에서 양성자-양성자 충돌 시 생성되는 수십억 개의 이벤트 중 극히 희귀한 물리 신호를 탐색하는 연구를 수행했습니다. 이 과정에서 체득한 방법론은 퀀트 투자 리서치와 구조적으로 동일합니다.

**1. 노이즈 속 신호 추출: Signal/Background → Alpha Signal/Noise**

물리학 실험에서 배경 이벤트 대비 0.01% 수준의 신호를 식별하는 것은, 시장 노이즈 속에서 미약한 알파 시그널을 포착하는 것과 본질적으로 같은 문제입니다. CNN 기반 분류기를 설계하여 기존 cut-based 방법론 대비 **신호 선별 효율을 1.85배 개선**했습니다. 이 경험은 비선형 패턴 인식과 피처 엔지니어링의 실전적 기반이 되었습니다.

**2. 대규모 데이터 처리: HPC 분산 학습 → 금융 빅데이터 파이프라인**

페타바이트급 충돌 데이터를 1024-node HPC 클러스터에서 분산 처리하며, 대규모 데이터 파이프라인의 설계와 최적화를 경험했습니다. 데이터 전처리, 배치 스케줄링, 병렬 학습 아키텍처에 대한 실전 지식은 금융 빅데이터 인프라 구축에 직접적으로 적용 가능합니다.

**3. 체계적 불확실성 정량화 → 모델 리스크 관리**

입자물리에서는 모든 측정값에 대해 통계적 불확실성(statistical uncertainty)과 체계적 불확실성(systematic uncertainty)을 엄밀하게 분리·정량화합니다. 검출기 보정 오차, 이론 모델 의존성, 시뮬레이션-실측 간 괴리 등 수십 개의 불확실성 원천을 개별적으로 평가하고, 이를 최종 결과에 전파(propagation)하는 훈련을 받았습니다. 이는 금융에서 모델 리스크를 정량화하고, 과적합 위험을 체계적으로 관리하는 역량으로 직결됩니다.

**4. 시뮬레이션 검증 → 백테스팅/아웃오브샘플 검증**

몬테카를로 시뮬레이션으로 생성한 예측과 실제 실험 데이터를 비교·검증하는 과정은, 백테스팅 결과를 아웃오브샘플 데이터와 라이브 성과로 검증하는 퀀트의 워크플로우와 구조적으로 동일합니다. 시뮬레이션과 실측의 불일치를 진단하고 보정하는 경험은 과적합에 대한 경계심과 검증 문화를 체화하게 해주었습니다.
