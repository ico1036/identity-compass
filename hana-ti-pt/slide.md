## 장 2: 대학원 — ML 선도 + CERN 연구 성과

### 내용

**학과 최초 ML/DL 도입 → CERN LHC 실험**

**ML 선도 (변곡점 ①의 실행)**
- 2017년, 물리학과 내 **최초** ML/DL 적용 제안
- 소규모 학회에서 먼저 결과 증명 → 교수 설득 → **GPU 인프라 도입**
- KISTI 국가슈퍼컴퓨팅센터 **Nurion 5호기** 활용 대규모 공동연구로 확장
- 우수논문상 2회 수상 — 외부 검증

**CERN LHC 실험**
- 수십억 충돌 이벤트 중 **0.01%** 희귀 신호 탐색
- CNN 기반 분류기 → 기존 방법론 대비 신호 선별 효율 **1.85배 개선**
- 페타바이트급 데이터, 1024-node HPC 클러스터 분산 처리

**이 경험이 퀀트에 가져온 것:**
```
물리학                          퀀트 리서치
─────────────────────────────────────────────
신호/배경 분리 (0.01%)     →   노이즈 속 알파 시그널 포착
시뮬레이션 vs 실측 비교     →   백테스트 vs 라이브 괴리 추적
체계적 불확실성 정량화      →   과적합 방어, 모델 리스크 관리
대규모 데이터 분산처리      →   대규모 전략 병렬 생성·검증
```

이 방법론을 들고 자산운용사에 갔다.

## 장 4: Level 3 — Agentic Alpha Factory

### 내용

**Level 3: 전략 가설 생성까지 자동화 — Team Lead**

**왜 이 구조인가 — 이론적 필연:**
> "영원히 황금알을 낳는 거위는 없다" — 짐 사이먼스

- 알파 = 시장 불균형. 균형 회복 시 소멸하는 것이 본질
- 궁극의 알파 하나에 올인 → 비효율. 어차피 decay
- **평균 이상의 약한 알파를 대량 생성 → 합성(Mega Alpha)이 정답** (Finding Alpha, 포트폴리오 이론)

**왜 지금 가능한가 — 비즈니스 근거:**
- 토큰 비용 vs 퀀트 인력 연봉의 비대칭 → 소규모 팀도 대량 알파 생산 가능
- 수만 알파 생성·합성은 글로벌 HF만의 해자였음 → **에이전트가 이 해자를 민주화**
- 글로벌 AI 경쟁 심화 = 잠식이 아니라 **알파 마이닝 성능 향상** (모델이 좋아질수록 수혜)

**Alpha Synthesis 구조:**
```
  [약한 α₁] [약한 α₂] [약한 α₃] ... [약한 αₙ]     ← 일 500+ 자동 생성
       \         |         |         /
        \        |         |        /
         ▼       ▼         ▼       ▼
      ┌──────────────────────────────┐
      │      Alpha Synthesis          │
      │  (합성 · 팩터 중립화 · 평가)   │
      └──────────┬───────────────────┘
                 │
                 ▼
          [ 강한 시그널 ]
```

**기술적 실현 — Harness Engineering:**
- ClaudeCode 기반 멀티에이전트 + Self-reflection Ralph Loop
- 에이전트 역할 분담: 퀀트 리서처 → 퀀트 개발자 → 백테스트 → 결과 분석가
- 목표 달성까지 루프가 자율 반복 — 할루시네이션은 교차 검증·TDD로 아키텍처 수준 방어

**성과 증명:**
- 이 루프를 Pod 형태로 병렬 구성 → **일 500+ 아이디어 자동 백테스트**
- 현재까지 가격 기반만으로 **Sharpe 1.3 이상 합성 알파 생성**

**Helix — 실행력의 증거:**
- Level 3를 독립 헤지펀드 구조로 확장 추진 (SG 프롭 + 알파 라이센싱)
- 설계·개발·사업화를 1인이 직접 실행 — 이 역량을 그룹 차원에서 발휘하고 싶다

---

**그룹 데이터 × Agentic Factory:**
- 현재까지는 가격 데이터 단일 소스 → 그럼에도 Sharpe 1.3+ 합성 알파
- 은행·증권·보험 — 서로 다른 도메인의 데이터가 합류하면?
- 가격만으로는 불가능했던 **orthogonal 알파**가 생성되고, **factor neutralization**의 축이 근본적으로 늘어남
- 수십만 약한 알파 중 서로 상관없는 것들만 골라 합성하는 구조가 비로소 완성됨
