# v020: 변곡점 감지 → 즉시 실행

## 원천
하나금융TI 자소서 + PT 슬라이드에서 본인이 직접 정의한 **핵심 역량 #2**:
> "기술 트렌드의 변곡점을 감지하면 즉시 방향을 트는 실행력"

## 근거
- 2017: DL 붐 초기, 학과 최초 ML 도입 → 논문상 2회
- 2024: LLM/Agent 붐 감지 → AI-native 리서치 전면 전환 → 기관 MOU

두 번 감지하고 두 번 실행으로 증명. 패턴이 아니라 정체성.

## 벡터
- 자율↔구조: **+0.85** (남이 안 할 때 먼저 움직임)
- 깊이↔폭: **+0.5** (감지는 폭, 실행은 깊이 — 중간)
- 혁신↔안정: **+0.95** (변곡점 자체가 혁신 시그널)
