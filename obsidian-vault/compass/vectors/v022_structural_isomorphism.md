# v022: 구조적 동형성 발견 능력

## 원천
자소서 핵심 역량 #1 (본인 정의):
> "서로 달라 보이는 문제에서 같은 구조를 발견하는 것"

## 근거
- 물리학 신호/배경 분리 → 퀀트 알파 시그널 포착
- 시뮬레이션-실측 비교 → 백테스트-라이브 괴리 추적
- 자화 모델 → Identity Compass (의사결정 추적)
- 도메인을 관통하는 추상화 = 이 사람의 사고방식

## 벡터
- 자율↔구조: **+0.7** (자기만의 프레임으로 문제를 재정의)
- 깊이↔폭: **+0.9** (본질/구조 추구 = 극단적 깊이)
- 혁신↔안정: **+0.7** (기존 구조를 새 도메인에 적용 = 응용 혁신)
