# Evidence-First Communication

Decision-ready language backed by evidence, not abstract theory. Core strength of [[Jiwoong Kim]].

## Communication Style
- Sensitivity/impact plots for uncertainty communication
- Probability ranges, not binary yes/no
- Stress tests across regimes for [[Model-Robustness-Testing]]
- High candor under deadlines
- Short, concrete updates with immediate progress visibility

## Principles
- [[Probabilistic-Decision-Making]]: frame everything probabilistically
- Kill-rules before deployment (from [[Signal-Validation]])
- Evidence over opinion, always
- Adaptive market-view updates from new evidence
